# Company Settings Module - Structure

## File Organization

```
src/
├── components/
│   └── company-settings/
│       ├── CompanyDashboard.tsx    # Main dashboard
│       ├── CompanyForm.tsx          # Create/edit form
│       ├── CompanyCard.tsx          # Company card display
│       └── CompanySearch.tsx         # Search component
└── app/
    └── api/
        └── companies/               # Company APIs
```

## Components

- CompanyDashboard - Main container
- CompanyForm - Create/edit form
- CompanyCard - Individual company display
- CompanySearch - Search functionality
